HTML & CSS

In this page I will show you what HTML and CSS is and how to use it.

1. HTML

HTML(Hyper Text Markup Language) is the standard markup language for creating Web pages.

1.1. How it works?

The HTML Language is written with < and >, and between those two symbols you write the actual tag and it's arguments.

For example, to show a paragraph we use <p> and to close the paragraph we use </p>.
