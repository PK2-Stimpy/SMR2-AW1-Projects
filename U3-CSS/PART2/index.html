1. HTML

HTML(Hyper Text Markup Language) is the standard markup language for creating Web pages.

1.1. How it works?

The HTML Language is written with < and >, and between those two symbols you write the actual tag and it's arguments.

For example, to show a paragraph we use <p> and to close the paragraph we use </p>.


You can also merge other languages with HTML like JavaScript, CSS and PHP.

1.2. Basic structure

The structure of a HTML document always starts with <!DOCTYPE html> to tell the browser it's a HTML document. Then, we open a <html> tag and inside this tag we have the <head> tag and the <body> tag.

The head tag is the part of the code that is ran before the body and is not visible by the user. You should put here scripts, metadata, and stuff like that...

The body tag is the part of the code where the actual content of the page goes.

1.2.1. Basic head tags

Some of the basic head tags would be the <title> tag, you specify a title for your page.

The <script> tag to define a head script.

The <meta> tag to define metadata of the page.

1.2.2. Basic body tags

We have <header> to define a header block and <footer> to define a footer block.

To include all the links we use <nav> so the user can navigate.

We use <section> to define big sections of content, <article> to delimit a specific content and <aside> to relate content of the page separately.

We use <hgroup> for header sections.

<figure> is used for multimedia, <time> for defining the time of the content and <mark> to define outstanding texts by reference.

We have <link /> for defining external resources.

<style> for defining a CSS style for the page.

From <h1> to <h6>, we have titles, with <blockquote> you insert bleeding on the text.

<p> is a paragraph, <br> a line jump and <hr> a visible line jump.

2. CSS

Cascading Style Sheets (CSS) is a stylesheet language used to describe the presentation of a document written in HTML.

CSS describes how elements should be rendered on screen, on paper, in speech, or on other media.

2.1. Introduction

This is a example style to make a paragraph red with the background color black.

            p {
        color: red;
        background-color: black;
    }

p is the selector.

color is a property of the selector.

red is the value of the property.


The selector can be:

  • A html tag.
  • A html class (if it starts with a dot)
  • A html id (if it starts with #)

You can combine selectors like this:

div .class {}
.class .class2 {}
#id .class {}
div #id {}

You can also do comments like this:

/* COMMENT */

2.2. Properties

Here are some of the basic properties that you can use in CSS.

You can also use the inline css properties explained in the html tutorial.


2.2.1. Color

The color property is for setting the color of the background.

Example:

This text will be red.

                p {
    color: red;
}

This text will be blue.

                p {
    /*     rgb(RED, GREEN, BLUE) */
    color: rgb(0  , 0    , 255);
}

2.2.2. Background Color

The background-color property is for setting the color of the background.

Example:

This text's background will be red.

                p {
    background-color: red;
}

This text's background will be blue.

                p {
    /*                rgb(RED, GREEN, BLUE) */
    background-color: rgb(0  , 0    , 255);
}

2.2.3. Background Image

The background-image property is for setting and image as the background.

Example:

This text has a background image and will be black. 

                p {
    background-image: url("https://www.lamp.es/images/54790/facebook.jpg");
    color: black;
}

2.2.4. Border

The border property is for setting a border.

Example:

This text will have a solid border.

                p {
    border: solid;
}

2.2.5. Font Family

The font-family property is for setting the font of the text.

Example:

This text will have the Segoe UI font.

                p {
    font-family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif;
}

2.2.6. Font Style

The font-style property is for setting the font style of the text.

Example:

This text will be italic.

                p {
    font-style: italic;
}

This text will be oblique.

                p {
    font-style: oblique;
}

2.2.7. Font Size

The font-size property is for setting the font size of the text.

Example:

This text will have 20px of font size.

                p {
    font-size: 20px;
}

2.2.8. Text Decoration

The text-decoration property is for decorating the text.

Example:

This text will have overline decoration.

                p {
    text-decoration: overline;
}

This text will have underline decoration.

                p {
    text-decoration: underline;
}

2.2.9. Margins

The margin property is for setting the margins of the object.

Example:

This text will have 70px of left margin.

                p {
    /* LEFT MARGIN */
    margin-left: 40px;

    /* RIGHT MARGIN */
    margin-right: 40px;

    /* TOP MARGIN */
    margin-top: 40px;

    /* BOTTOM MARGIN */
    margin-top: 40px;
}
